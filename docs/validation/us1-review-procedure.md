# US1 Review Procedure

## Purpose

Guide maintainers through acceptance of the foundation baseline.

## Review Steps

1. Confirm `README.md` states educational/non-advisory purpose.
2. Confirm `AGENTS.md` references the current plan and guardrails.
3. Confirm artifact paths match `docs/foundation/artifact-manifest.md`.
4. Confirm no runtime application behavior is introduced.
5. Confirm validation evidence is included in the PR.

## Acceptance

US1 passes when a reviewer can complete the checklist without external clarification.
